# grep.core
Intelligent grep in .NET Core

## Planned features
* Console & UI
* Cross platform
* Multi-threaded
* Performance stats
* Regex/plain/wildcard modes for search & path
* Intelligent file inclusion/exclusion
  * By content
  * By extension
  * By ignore file
* Pluggable result interpretation
  * Parse source files for context
  * Render markdown/html/etc.
